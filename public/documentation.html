AFK Bot Plugin Documentation
Overview

The AFK Bot allows you to create and manage automated Minecraft bots with plugin support.

Getting Started

To install dependencies, run:

npm install mineflayer express socket.io fs path
Plugin Structure

A plugin must have a folder inside the /plugins directory with:

  • plugin.json - Metadata
  • index.js - Main logic
Example Plugin

Here's a simple plugin that sends a message when the bot joins:

module.exports.init = (bot) => {
    bot.on('spawn', () => {
        bot.chat("Hello! I am an AFK bot.");
    });
};
Installing a Plugin

To install a plugin, place it in the /plugins directory and add it to the bot config:

{
    "username": "Bot1",
    "server": "your.server.ip",
    "port": 25565,
    "version": "1.20.1",
    "enabledPlugins": ["yourPlugin"]
}
API Endpoints

Start a bot using:

curl -X POST http://localhost:3001/api/bots/start -H "Content-Type: application/json" -d '{"username":"Bot1"}'

Stop a bot:

curl -X POST http://localhost:3001/api/bots/stop -H "Content-Type: application/json" -d '{"username":"Bot1"}'
